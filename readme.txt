Put a widget treeview to list all articles in your website

== Description ==

With WPArchivesTree you can display a widget which list all articles in your website on a treeview.
Articles are group by month and year and you can choose if you want to display the first month articles or not.

== Installation ==

1. Upload wparchivestree to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place the widget in the 'Widgets' menu in WordPress
